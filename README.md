## 特性

- 基于 karma 实现, 运行测试于浏览器端;
- 内置 es2015, react, stage-0;
- 内置 mocha, chai, sinon 等常用测试工具类库;
- 更专注于写测试用例本身;

## 全局API

- [mocha](http://mochajs.org/)
- [chai](http://chaijs.com/api)
- [sinon](http://sinonjs.org/)
- 持续更新中...

## 使用

> 约定: 测试文件的文件名加 -test

```
"srcipts": {
  "test": "atool-test"
}

```
